⚡ Quick Installation

1. Install A-Coder CLI

pip install a-coder-cli

2. Create Configuration

cp config.json.example config.json
# Edit config.json with your API key and model settings

3. Run in Your Project

# Navigate to your project
cd /path/to/your/project

# Start A-Coder
a-coder --config ~/config.json

# Or use the short form:
a-coder -c ~/config.json

That's it! The filesystem server automatically configures itself to your current directory.

💡 Key Commands

/add <filepath>
Add file to conversation context (editable)
/add-ro <filepath>
Add file as read-only reference
/files
List all added files with status
/clear-files
Clear all added files from context
/mcp-list
List connected MCP servers
/help
Show detailed help information
